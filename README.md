## Title: Using LiDAR and Satellite Data for Plant Height and Biomass Estimation

### Data Sources:
- GEDI LiDAR canopy height (NASA)
- Global Biomass dataset (ESA CCI)

- Build ML model for height–biomass relationships.

### Output
